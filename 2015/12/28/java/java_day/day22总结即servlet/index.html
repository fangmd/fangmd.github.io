##Servlet

иҝҗиЎҢеңЁжңҚеҠЎеҷЁз«Ҝзҡ„web serverдёӯзҡ„javaе°ҸзЁӢеәҸ

web server:webжңҚеҠЎеҷЁпјҢеёёи§Ғзҡ„жңүпјҡtomcatгҖҒjbossгҖҒwebsphereгҖҒiisзӯүjspеј•ж“Һ

tomcatгҖҒjbossпјҡзәҜзІ№дҪҝз”ЁjavaиҜӯиЁҖзј–еҶҷзҡ„пјҢе°ҶжңҚеҠЎз«Ҝ(ServerSocket)зҡ„е…ұйҖҡзҡ„д»Јз ҒжҠҪеҸ–еҮәжқҘпјҢеҒҡжҲҗдёҖдёӘдә§е“ҒгҖӮ

зҪ‘з«ҷдёҺweb serverзҡ„е…ізі»пјҢwebжңҚеҠЎеҷЁдёӯеҸҜд»ҘйғЁзҪІnдёӘзҪ‘з«ҷгҖӮ

Servletзҡ„з”ҹе‘Ҫе‘ЁжңҹпјҡвҖ“>еә•еұӮдҪҝз”ЁдәҶеҚ•дҫӢи®ҫи®ЎжЁЎејҸ еҮәз”ҹвҖ“жҸҗдҫӣжңҚеҠЎ(deGet(),doPost())вҖ“й”ҖжҜҒ(жңҚеҠЎеҷЁйҮҚеҗҜпјҢе…ій—ӯ)

url:з»ҹдёҖиө„жәҗе®ҡдҪҚз¬ҰгҖӮз”ЁжқҘж ҮзӨәдә’иҒ”зҪ‘дёҠиө„жәҗзҡ„дҪҚзҪ®гҖӮ(http://xxx)

uri:з»ҹдёҖиө„жәҗж ҮиҜҶз¬ҰгҖӮurlжҳҜuriзҡ„еӯҗйӣҶгҖӮ(ftp://xxx)

###Servletзҡ„и·іиҪ¬ж–№ејҸ

  1. е®ўжҲ·з«Ҝи·іиҪ¬

    1. ең°еқҖж ҸеҸ‘з”ҹж”№еҸҳ
    2. дёҚдјҡдј йҖ’requestеҜ№иұЎ
    3. response.sendRedirect(вҖңең°еқҖвҖқ)

  2. жңҚеҠЎз«Ҝи·іиҪ¬

    1. ең°еқҖж ҸдёҚдјҡеҸ‘з”ҹеҸҳеҢ–
    2. иғҪдј йҖ’requestеҜ№иұЎ
    3. request.getRequestDispatcher(вҖңзӣ®зҡ„йЎөйқўең°еқҖвҖқ).forward(resp,req);

йӘҢиҜҒпјҡжЁЎжӢҹз”ЁжҲ·зҷ»е…ҘеҲ°ж–°жөӘйӮ®з®ұзҡ„жғ…еҪў

йЎөйқўпјҡ

  1. зҷ»е…Ҙз•Ңйқў(html)
  2. зҷ»е…ҘжҲҗеҠҹз•Ңйқў
  3. зҷ»е…ҘеӨұиҙҘз•Ңйқў

###еӯ—з¬ҰйӣҶд№ұз Ғи§ЈеҶі
иЎЁеҚ•жҸҗдәӨж–№ејҸдёҚеҗҢпјҢеӯ—з¬ҰйӣҶеӨ„зҗҶд№ұз Ғй—®йўҳзҡ„и§ЈеҶіж–№жЎҲд№ҹдёҚеҗҢ

  1. getжҸҗдәӨж–№ејҸ

    1. ж–№жЎҲдёҖпјҡзЎ¬зј–з Ғж–№ејҸвҖ“еёёз”Ёж–№ејҸдҪҶжҳҜдёҚжҺЁиҚҗпјҡ

      req.setCharacterEncoding("UTF-8");
String name = req.getParameter("name");//iso-8859-1
String name = new String(name.getBytes("iso-8859-1"),"UTF-8");

    2. ж–№жЎҲдәҢпјҡй…ҚзҪ®ж–Үд»¶дҝ®ж”№ж–№ејҸпјҲжҺЁиҚҗпјү

      <Connector connectionTimeout="20000" URIEncoding="UTF-8"

      disableUploadTimeout=вҖқfalseвҖқ executor=вҖқtomcatThreadPoolвҖқ port=вҖқ8080вҖқ protocol=вҖқHTTP/1.1вҖқ redirectPort=вҖқ8443вҖқ/>

  2. postжҸҗдәӨж–№ејҸ

    req.setCharacterEncoding("UTF-8");
response.setContentType("text/html;charset=UTF-8");

    ###servletжңҚеҠЎеҷЁз«Ҝж¶үеҸҠзҡ„api

  3. HttpServletзұ»
    1. doGet(HttpServletRequest req, HttpServletResponse resp)
    2. dePost(HttpServletRequest req, HttpServletResponse resp)

жңҚеҠЎеҷЁзј–зЁӢжӯҘйӘӨпјҡ

###servletе®ўжҲ·з«Ҝзј–зЁӢж¶үеҸҠзҡ„api
URLзұ»пјҡд»ЈиЎЁдёҖдёӘз»ҹдёҖиө„жәҗе®ҡдҪҚз¬ҰпјҢжҢҮеҗ‘дә’иҒ”зҪ‘иө„жәҗзҡ„жҢҮй’Ҳ

  1. URL(String spec):йҖҡиҝҮеҸӮж•°жҢҮе®ҡзҡ„urlең°еқҖжһ„е»әеҜ№иұЎ
  2. URLConnection openConnection():жһ„е»әе®ўжҲ·з«ҜеҲ°жңҚеҠЎеҷЁиҝһжҺҘзҡ„еҜ№иұЎ

URLConnectionзұ»пјҡд»ЈиЎЁеә”з”ЁзЁӢеәҸе’ҢURLд№Ӣй—ҙзҡ„йҖҡдҝЎй“ҫжҺҘ

  1. InputStream getInputStream():д»ҺзҪ‘з»ңдёҠиҺ·еҫ—ж•°жҚ®(зҪ‘з»ңеҲ°жң¬ең°е®ўжҲ·з«Ҝзҡ„йҖҡи·Ҝ)
  2. OnputStream getOutputStream():е®ўжҲ·з«Ҝеҗ‘иҝңзЁӢжңҚеҠЎеҷЁдј йҖ’ж•°жҚ®(е®ўжҲ·з«ҜеҲ°зҪ‘з»ңзҡ„йҖҡи·Ҝ)
  3. void setDoInput(boolean doinput):е°ҶжңҚеҠЎеҷЁеҲ°е®ўжҲ·з«Ҝзҡ„йҖҡи·Ҝи®ҫзҪ®дёәй—ӯеҗҲжҲ–иҖ…йҖҡз•…(й»ҳи®ӨжҳҜtrue)
  4. void setDoOuput(boolean doouput):е°Ҷе®ўжҲ·з«ҜеҲ°жңҚеҠЎеҷЁзҡ„йҖҡи·Ҝи®ҫзҪ®дёәй—ӯеҗҲжҲ–иҖ…йҖҡз•…(й»ҳи®ӨжҳҜfalse)

URLConnectionзҡ„еӯҗзұ»HttpURLConnection:

  1. еёёз”Ёеӯ—ж®ө
    1. static int HTTP_OK:е®ўжҲ·з«ҜеҸ‘йҖҒзҡ„ж•°жҚ®жңҚеҠЎеҷЁе·Із»ҸеӨ„зҗҶе®ҢжҜ•пјҢе№¶з»ҷеҮәдәҶжӯЈеёёзҡ„еҸҚйҰҲ
    2. static int HTTP_NOT_FOUND:е®ўжҲ·з«ҜиҜ·жұӮзҡ„жңҚеҠЎз«Ҝзҡ„иө„жәҗдёәжүҫеҲ°

йңҖжұӮпјҡд»Һжң¬ең°жңҚеҠЎеҷЁдёӢиҪҪе’ҢдёҠдј еӣҫзүҮ

еүҚжҸҗпјҡ

  1. еңЁжң¬ең°жңҚеҠЎеҷЁзӣёеә”зҪ‘з«ҷдёӢеҮҶеӨҮиө„жәҗ
  2. еҗҜеҠЁжңҚеҠЎеҷЁпјҢдҪҝе…¶еӨ„дәҺиҝҗиЎҢзҠ¶жҖҒ

е®ўжҲ·з«Ҝзј–зЁӢжӯҘйӘӨпјҡ

  1. е°ҶзҪ‘еқҖе°ҒиЈ…жҲҗURLеҜ№иұЎ
  2. жһ„е»әиҝһжҺҘиҺ·еҸ–еҜ№иұЎHttpURLConnection
  3. иҺ·еҸ–зҪ‘з»ңеӯ—иҠӮиҫ“е…ҘжөҒеҜ№иұЎ
  4. жһ„е»әFileOutputStreamеҜ№иұЎ(е°ҶеӣҫзүҮдёӢиҪҪеҲ°жң¬ең°)
