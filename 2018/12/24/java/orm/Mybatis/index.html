Mybatis 与 Hibernate 区别

Mybatis 优点：

Mybatis: 需要自己写 sql 语句。
Mybatis 自己控制 sql 语句，方便 sql 优化。

Mybatis 缺点：

Mybatis: 无法快速切换数据库，sql 语句与数据库相关，Hibernate 数据库无关可以随时切换。

Mybatis 实现 Dao 层

动态代理，Mapper 接口类遵循四个原则:

  1. 接口方法名称与 xml 中的id一致
  2. 返回值类型与 xml 中一致
  3. 方法的入参类型与 xml 中一致
  4. 命名空间绑定此接口
