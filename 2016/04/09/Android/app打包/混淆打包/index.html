代码混淆

语法

  1. 自定义的实体类不能被混淆，保证一个包下的所有类不被混淆：

    -keep class 你的实体类所在的包.** { *; }

  2. 使用第三方包的时候，需要设置第三方包提供的混淆内容：

    有第三方提供

  3. 与js互调的类

    -keep class 你的类所在的包.** { *; }
-keepclasseswithmembers class 你的类所在的包.父类$子类 { <methods>; } ＃ 如果是内部类

  4. 反射相关的类

    -keep class 你的类所在的包.** { *; }
