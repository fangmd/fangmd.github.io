什么是html

用来描述网页的语音。

  • HTML指的是超文本标记语言(Hyper Text Markup Language)
  • HTML不是一种编程语言，而是一种标记语言(markup language)
  • 标记语言是一套标记标签(markup tag)
  • HTML使用标记标签来描述网页

HTML标签

  • 由尖括号包围的关键词
  • 成对出现
  • 开始标签和结束标签
  • 开始和结束也称为开放标签和闭合标签

HTML文档 = 网页

Web游览器作用是读取HTML文档，并以网页的形式显示他们。

基本HTML标签 Tag

  • 标题：<h1>-<h6>
  • 段落：<p>
  • 链接：<a>,在 href 属性中指定链接的地址
  • 图像：<img>,在属性中指定图像的名称，尺寸
  • 与 之间的文本描述网页
  • 与 之间的文本是可见的页面内容
