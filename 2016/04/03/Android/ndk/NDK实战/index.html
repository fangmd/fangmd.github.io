NDK 实例

简单Demo

参考：http://yanbober.github.io/2015/02/14/android_studio_jni_1/

  1. 创建一个类用于放置调c的代码：

    public class NdkJniUtils {

    public native String getCLanguageString();

    }

  2. 在Activity中使用这个方法

  3. 构建工程

  4. 在工程目录下：NDKApplication\app\build\intermediates\classes\debug根据class文件利用javah生成对应的.h头文件

    xxxxx\debug> javah -jni io.github.yanbober.ndkapplication.NdkJniUtils

  5. 执行完之后你可以在文件夹NDKApplication\app\build\intermediates\classes\debug下看见生成的 .h头文件

  6. 在工程的main目录下新建一个名字为jni的目录，然后将刚才的 .h文件剪切过来。 在jni目录下新建一个c文件，随意取名，我的叫jnitest.c 。编写代码：

    #include "io_github_yanbober_ndkapplication_NdkJniUtils.h"
/*
 * Class:     io_github_yanbober_ndkapplication_NdkJniUtils
 *  * Method:    getCLanguageString
 *   * Signature: ()Ljava/lang/String;
 *    */
 *    JNIEXPORT jstring JNICALL Java_io_github_yanbober_ndkapplication_NdkJniUtils_getCLanguageString
 *      (JNIEnv *env, jobject obj){
              return (*env)->NewStringUTF(env,"This just a test for Android Studio NDK JNI developer!");
        }
 *      }

  7. 接下来在工程的local.properties文件中添加NDK路径（上面下载好的那个NDK）

  8. 接下来在app module目录下的build.gradle中设置库文件名（生成的so文件名）。找到gradle文件的defaultConfig这项，在里面添加如下内容：

    defaultConfig {
    ......
    ndk{
        moduleName "YanboberJniLibName"            //生成的so名字
            abiFilters "armeabi", "armeabi-v7a", "x86"    //输出指定三种abi体系结构下的so库。目前可有可无。

    }

}

  9. 现在生成的so库名字也有了，那就去代码的NdkJniUtils java文件添加静态初始化load代码，添加如下：

    static {
        System.loadLibrary("YanboberJniLibName");    //defaultConfig.ndk.moduleName

}

  1. 将c文件编译成.so文件
    jni文件夹下创建：Android.mk:

    LOCAL_PATH := $(call my-dir)
    include $(CLEAR_VARS)
    LOCAL_MODULE := jnimodule
    LOCAL_SRC_FILES := jnitest.c
    include $(BUILD_SHARED_LIBRARY)

第四项改成自己的.c文件名，多个可以添加多个第四个语句；

jni文件夹下创建:Application.mk:

APP_ABI := armeabi armeabi-v7a x86 mips

构建：cd到jni目录

ndk-build

构建完成后会看到mian下增加的相应的编译文件；

  1. 配置gradle自动编译c文件

    android.ndk {
        // 模块名称
        moduleName = "hello-jni"

        // 指定编译平台，更多平台信息 参见https://developer.android.com/ndk/guides/abis.html#sa
        abiFilters "armeabi", "armeabi-v7a"
        /*
         * Other ndk flags configurable here are
     * cppFlags.add("-fno-rtti")
     * cppFlags.add("-fno-exceptions")
     * ldLibs.addAll(["android", "log"])
     * stl       = "system"
     */
     *

    }
