OkHttpClient.class

作用：

  1. 发送 http 请求
  2. 读取 http 请求结果

在一个工程中建议自己维护一个单例 OkHttpClient 对象，因为每个 OkHttpClient 对象都有自己的 连接池和线程池，每次请求都创建一个 OkHttpClient 对象会导致性能消耗过大。
