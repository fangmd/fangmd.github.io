个人体验

  1. 慢：报错慢，提示慢
  2. 错误提示体验很差
  3. 没有 AS 中的点语法 “.var .field”
  4. 没有 AS 中的模版语法
  5. 代码重构不方便

常用快捷键

  • 格式化代码： control + i
  • 快捷修复代码：
  • 清理工程缓存：CMD + shift + k (在 pod install 后如果第三方包 import 失败可以先清理一下工程)
  • 清理：command + shift + option + k
