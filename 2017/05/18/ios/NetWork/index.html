插件

Moya 在初始化 Provider 的时候可以传入一些插件，Moya库中默认有4个插件。

  • AccessTokenPlugin 管理AccessToken的插件
  • CredentialsPlugin 管理认证的插件
  • NetworkActivityPlugin 管理网络状态的插件
  • NetworkLoggerPlugin 管理网络log的插件
