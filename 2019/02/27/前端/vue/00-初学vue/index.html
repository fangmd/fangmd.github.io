  1. v-on 添加点击事件
  1. v-model 实现表单输入和应用状态之间的双向绑定

绑定的属性：

text, textarea: value, input
checkbox, radio: checked, change
select: value, change

修饰符:.lazy, .number, .trim

  1. 计算属性，结果基于属性值，有缓存比方法更好(在相关值没有改变的时候， 不需要计算直接返回)，不基于属性的计算不要使用这个函数

计算属性缓存 VS 方法
计算属性 VS 侦听属性

计算属性默认只有 get, 添加 set

  1. 使用 key 管理可复用的元素

html 内容改变的时候是否要重新渲染组件，通过设置不同的 key 实现强制重新渲染
重新渲染后：用户输入的内容会清空

  1. v-show 控制是否显示，和 display 一样，隐藏后在 html 源码中还是能找到

v-show：初始化开销大，切换开销小。适合切换频率高的内容

  1. v-once 创建低开销的静态组件(通常不使用，除非有性能问题)
